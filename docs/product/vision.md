# Product Vision

## Scopo

Diario Intelligente deve diventare una memoria tipo cervello per una persona.

L'utente scrive o parla in modo naturale. Il sistema:

- capisce cosa e successo
- estrae eventi, nodi e connessioni
- mantiene memoria nel tempo
- permette correzione e CRUD via chat
- restituisce chiarezza, pattern, report e prossimi passi

## North Star

Creare un motore cognitivo personale che aumenta nel tempo il valore della memoria dell'utente e la probabilita di raggiungere i suoi obiettivi.

## Cosa deve fare davvero

### 1. Memoria esterna affidabile

Registrare e recuperare:

- eventi
- pensieri
- decisioni
- persone
- progetti
- problemi
- obiettivi
- stati energetici o emotivi

### 2. Trasformare caos in struttura

L'utente non deve compilare form rigidi.

Il sistema deve partire da input libero e costruire:

- nodi
- relazioni
- timeline
- pattern ricorrenti

### 3. Far emergere pattern

Il valore non e solo nel logging. Il sistema deve aiutare a capire:

- cosa drena energia
- cosa produce progresso reale
- quali problemi si ripetono
- quali fattori anticipano regressione o blocco

### 4. Trasformare obiettivi in percorsi

Un obiettivo non e una riga in una todo list.

Un obiettivo e un nodo connesso a:

- sotto-obiettivi
- vincoli
- risorse
- ostacoli
- micro-step disponibili ora

### 5. Guidare l'azione quotidiana

Alla domanda "cosa faccio adesso?" il sistema deve proporre pochi passi concreti, non motivazione generica.

## Modello mentale

Tutto viene rappresentato come rete personale:

- nodi: goal, persone, progetti, abitudini, skill, task, emozioni, problemi, eventi, decisioni
- edge: dipendenza, causa, sequenza, correlazione, rinforzo, conflitto, ricorrenza

Nel tempo alcuni nodi diventano hub piu forti e alcuni percorsi diventano preferenziali.

## Output attesi

- daily summary
- weekly review
- spiegazione dei pattern
- ricerca conversazionale
- correzione via chat
- micro-step suggeriti
- mappa del percorso verso un obiettivo

## Non obiettivi

Il prodotto non deve diventare:

- un generico note app
- un clone di Notion
- un motivational coach astratto
- un sistema che chiede troppi input strutturati

## Principi di comportamento del sistema

- actionable over insightful
- smallest viable step first
- few choices, high clarity
- distinguere fatto da inferenza
- personalization via usage, non via configurazione pesante
